# PROJET WEB (Groupe la passerelle)

## Description

Site web professionnel présentant une entreprise avec ses services, son équipe et des opportunités de recrutement. Le site propose une interface moderne et responsive avec des formulaires interactifs.

## Fonctionnalités principales
- page debut (chargement de la page d'acceuil)
- Accueil : Page d'accueil 
- Présentation : Présentation de l'équipe
- Nos Expertises : différents domaines (BDD, Analyse de données, Programmation, Réseaux, Sécurité)
- Actualités : Aarticles actualisés
- Recrutement : Formulaire de candidature avec upload de CV validé
- Contact : Formulaire de contact pour les demandes de renseignements

## Formulaire de Recrutement
- Upload de CV (PDF, DOC, DOCX)
- Validation en temps réel des champs
- Drag & drop pour les fichiers
- Limite de taille : 5 MB

### Formulaire de Contact
- Validation email et téléphone
- Formatage automatique du numéro de téléphone
- Messages de confirmation
- Envoi sécurisé des données

### Navigation
- Menu responsive pour mobile et desktop
- Dropdown pour la section Expertise
- Navigation fluide avec scroll smooth

### Thème
- Basculement clair/sombre
- Sauvegarde des préférences utilisateur
- Thème personnalisable via CSS

## Points clés
- Validation côté client complète
- Code organisé
- Performance optimisée


### Le formulaire ne s'envoie pas
- Vérifier la connexion internet

### Le fichier ne s'upload pas
- Vérifier le format (PDF, DOC, DOCX uniquement)
- Vérifier la taille (max 5 MB)
- Essayer un autre navigateur
